Web UI Guide

The web interface lets you audit any website visually with real-time progress.

Analyzing a Website

  1. Enter the domain in the input field (e.g. example.com)
  2. Optionally, click "Customize Rules" to select specific rules or use a preset
  3. Click "Analyze" to start the crawl
  4. Watch real-time progress as pages are analyzed
  5. Review the full report when analysis completes

Using Rule Presets

Presets let you quickly select a group of rules relevant to your site type:

  • Full Scan — all 130+ rules for maximum coverage
  • Essential — core SEO checks every site needs
  • Blog / Article — optimized for content-heavy sites
  • E-Commerce — product pages, catalog, and transactional sites
  • Video Platform — video-centric sites (YouTube-like, courses)
  • Landing Page — single-page sites and marketing pages
  • Technical SEO — infrastructure, crawlability, and indexation only
  • GEO / AI Readiness — AI search engine optimization checks only

Customizing Rules

Click "Customize Rules" to expand the rules panel. You can:

  • Toggle entire categories on/off using category checkboxes
  • Select or deselect individual rules
  • Use "Select All" / "Deselect All" for quick changes
  • Apply a preset then fine-tune individual rules

Reading the Report

The report groups issues by category. Each issue group shows:

  • Severity badge (critical, high, medium, low, info)
  • Issue message — what's wrong
  • Affected URL count — how many pages are impacted
  • Recommendation — how to fix it
  • Affected URLs — clickable links to each affected page
  • Relevant assets — images, scripts, or other resources related to the issue

Excluding Rules from Export

On the report page, each issue group has a checkbox. Uncheck issues you want to exclude. Unchecked issues will be omitted from CSV and Markdown exports, but remain visible (dimmed) in the report.

Exporting Reports

Download reports in two formats:

  • CSV — spreadsheet-friendly format for manual review
  • Markdown — AI-agent-friendly format that can be used with coding assistants to automatically fix issues
