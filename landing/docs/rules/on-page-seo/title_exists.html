Page has a <title> tag

critical

title_exists

Checks if the page has a <title> tag in the <head> section.

What does this rule check?

The <title> tag is the single most important on-page SEO element. It defines the page's topic for search engines and appears as the clickable headline in search results. Every HTML page must have exactly one <title> tag inside the <head> element.

Why is this important?

Without a title tag, search engines cannot determine the page's topic, leading to poor rankings and unreadable search result listings. Users see a blank or URL-based entry in browser tabs and bookmarks.

How to fix

Add a <title> tag inside <head> with a unique, descriptive title for each page. Include the primary keyword naturally. Keep it between 30-60 characters.

Examples

Incorrect 
<head>
  <meta charset="UTF-8" />
  <!-- No title tag! -->
</head>
Correct 
<head>
  <meta charset="UTF-8" />
  <title>Best Running Shoes 2025 - Expert Reviews | ShoeHub</title>
</head>
