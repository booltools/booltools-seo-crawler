Image alt text is descriptive

low

images_alt_descriptive

Last updated: 2026-06-23

Checks that image alt text is descriptive, not generic.

What does this rule check?

Alt text should describe the image content meaningfully. Generic alt text like 'image', 'photo', 'img_001', or the filename provides almost no value for accessibility or SEO.

Why is this important?

Non-descriptive alt text defeats the purpose of having alt attributes. It provides no context for visually impaired users and wastes SEO opportunities.

How to fix

Replace generic alt text with specific descriptions of what the image shows. Include relevant keywords naturally.

Examples

Incorrect 
<img src="product.jpg" alt="image" />
<img src="hero.jpg" alt="photo" />
<img src="DSC_0001.jpg" alt="DSC_0001" />
Correct 
<img src="product.jpg" alt="Blue wireless headphones with noise cancellation" />
<img src="hero.jpg" alt="Team hiking through mountain trail at sunset" />
