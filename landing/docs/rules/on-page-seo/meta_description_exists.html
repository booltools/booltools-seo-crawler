Page has a meta description

high

meta_description_exists

Last updated: 2026-06-24

Checks if the page has a meta description tag.

What does this rule check?

The meta description provides a summary of the page content. While not a direct ranking factor, it heavily influences click-through rate from search results. Search engines display this as the snippet below the title in results pages.

Why is this important?

Without a meta description, search engines auto-generate snippets from page content, which may not be compelling or relevant. A well-crafted description can significantly improve CTR from search results.

How to fix

Add a meta description tag with a compelling, action-oriented summary of the page content. Include the target keyword naturally.

Examples

Incorrect 
<head>
  <title>Product Page</title>
  <!-- No meta description -->
</head>
Correct 
<head>
  <title>Product Page</title>
  <meta name="description" content="Shop the latest running shoes with free shipping. Compare prices, read reviews, and find the perfect pair for your next marathon." />
</head>
