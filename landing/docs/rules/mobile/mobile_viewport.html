Rules / Mobile / Viewport meta tag exists

Viewport meta tag exists

critical

mobile_viewport

Checks if the viewport meta tag exists.

What does this rule check?

The viewport meta tag is essential for mobile rendering. Without it, mobile browsers default to a desktop-width viewport and scale down the page, making it unusable on small screens.

Why is this important?

No viewport tag means the page is not mobile-friendly. Google uses mobile-first indexing, so non-mobile-friendly pages rank significantly lower.

How to fix

Add the viewport meta tag to every page's <head> section.

Examples

Incorrect 
<head>
  <!-- No viewport tag - page renders at desktop width on mobile -->
</head>
Correct 
<head>
  <meta name="viewport" content="width=device-width, initial-scale=1" />
</head>
