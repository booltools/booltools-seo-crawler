twitter:card meta tag is present

medium

twitter_card

Checks if the twitter:card meta tag is present.

What does this rule check?

The twitter:card tag defines the card type for Twitter/X link previews. Options are 'summary', 'summary_large_image', 'app', or 'player'.

Why is this important?

Without a twitter:card tag, Twitter may not display rich link previews, reducing engagement with shared links.

How to fix

Add a twitter:card meta tag. Use 'summary_large_image' for most content pages.

Examples

Incorrect 
<!-- No Twitter card tags -->
Correct 
<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Page Title" />
<meta name="twitter:description" content="Description" />
<meta name="twitter:image" content="https://example.com/image.png" />
