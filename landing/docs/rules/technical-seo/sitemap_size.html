Sitemap is within size limits

medium

sitemap_size

Last updated: 2026-06-24

Sitemap is within size limits. This audit rule is part of the Booltools SEO Crawler rule set for optimizing search engine visibility.

What does this rule check?

This rule verifies that your page follows the "Sitemap is within size limits" best practice for search engine optimization and generative engine optimization (GEO). It is checked automatically by the Booltools SEO Crawler.

Why is this important?

Following the "Sitemap is within size limits" best practice improves your page's visibility in search engines and AI-powered search systems. Failing this check can negatively impact your SEO score.
