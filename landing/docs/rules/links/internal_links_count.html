Rules / Links / Internal link count is reasonable

Internal link count is reasonable

low

internal_links_count

Last updated: 2026-06-23

Checks that the internal link count is reasonable.

What does this rule check?

Pages should have a reasonable number of internal links. Too few means poor navigation. Too many can dilute link equity and overwhelm users.

Why is this important?

An excessive number of links on a single page dilutes the value passed to each linked page. Too few links means missed navigation and SEO opportunities.

How to fix

Keep internal links focused and relevant. Remove redundant or unnecessary links. Aim for a natural number based on content length.

Examples

Incorrect 
<!-- Page with 500+ internal links in a massive footer -->
Correct 
<!-- Page with 20-50 relevant internal links in content, navigation, and sidebar -->

Review and optimize internal link count:
1. Remove duplicate links to the same URL
2. Remove low-value links (e.g., "click here")
3. Keep navigation links reasonable
4. Focus on contextual links within content
