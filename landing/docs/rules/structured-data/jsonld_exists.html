Page has JSON-LD structured data

medium

jsonld_exists

Checks if the page has JSON-LD structured data.

What does this rule check?

JSON-LD (JavaScript Object Notation for Linked Data) is the recommended format for structured data. It helps search engines understand page content and can enable rich snippets in search results.

Why is this important?

Without structured data, search engines cannot generate rich snippets (star ratings, prices, FAQs, etc.) which significantly improve click-through rates.

How to fix

Add JSON-LD structured data relevant to your content type (Article, Product, FAQ, Organization, etc.).

Examples

Incorrect 
<head>
  <!-- No structured data -->
</head>
Correct 
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "How to Train for a Marathon",
  "author": {"@type": "Person", "name": "Jane Smith"}
}
</script>
