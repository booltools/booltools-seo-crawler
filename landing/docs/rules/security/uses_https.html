Page uses HTTPS

critical

uses_https

Last updated: 2026-06-23

Checks if the page is served over HTTPS.

What does this rule check?

HTTPS encrypts data between the browser and server, protecting user privacy and data integrity. It has been a ranking signal since 2014 and is required for modern web features like service workers and HTTP/2.

Why is this important?

HTTP pages are marked as 'Not Secure' by browsers, destroying user trust. Google explicitly uses HTTPS as a ranking signal. Many browser features require HTTPS.

How to fix

Install an SSL certificate and redirect all HTTP traffic to HTTPS.

Examples

Incorrect 
http://example.com
<!-- Not secure, browser shows warning -->
Correct 
https://example.com
<!-- Secure, encrypted connection -->
