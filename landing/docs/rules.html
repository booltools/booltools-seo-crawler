On-Page SEO (15 rules)

Title tags, meta descriptions, heading hierarchy, image alt text, and content optimization.

  • title_exists critical — Page has a <title> tag
  • title_length medium — Title is between 30-60 characters
  • title_multiple high — Page has only one title tag
  • meta_description_exists high — Page has a meta description
  • meta_description_length medium — Description is 120-160 characters
  • meta_description_multiple medium — Page has only one meta description
  • h1_count high — Page has exactly one H1
  • h1_not_empty high — H1 has non-empty text
  • heading_hierarchy medium — Headings follow proper hierarchy
  • heading_not_empty low — Headings have non-empty text
  • images_alt_text high — All images have alt attributes
  • images_alt_descriptive low — Image alt text is descriptive
  • images_dimensions medium — Images have width/height attributes
  • images_modern_format low — Images use WebP/AVIF format
  • images_lazy_loading low — Below-fold images use lazy loading

Content (2 rules)

Word count and text-to-HTML ratio.

  • content_word_count medium — Page has sufficient word count
  • content_text_html_ratio medium — Text to HTML ratio is healthy

Technical SEO (27 rules)

Canonical URLs, robots directives, sitemaps, redirects, and HTTP status codes.

  • canonical_exists high — Page has a canonical link
  • canonical_absolute medium — Canonical URL is absolute
  • canonical_self_ref low — Canonical is self-referencing
  • canonical_conflict high — No conflicting canonical tags
  • meta_robots_noindex info — Page has a noindex directive
  • meta_robots_conflict high — No conflicting robots directives
  • http_status_ok critical — Page returns HTTP 200 status
  • crawl_depth medium — Page crawl depth is reasonable
  • pagination_rel_tags low — Pagination uses rel=next/prev
  • robots_txt_exists high — robots.txt file exists
  • robots_txt_size low — robots.txt is not too large
  • robots_txt_sitemap medium — robots.txt references sitemap
  • robots_txt_syntax medium — robots.txt has valid syntax
  • sitemap_exists high — Sitemap.xml exists
  • sitemap_valid_xml high — Sitemap has valid XML
  • sitemap_index info — Sitemap index detected
  • sitemap_size medium — Sitemap is within size limits
  • sitemap_freshness low — Sitemap URLs have recent lastmod
  • sitemap_coverage medium — Crawled URLs appear in sitemap
  • sitemap_orphan_urls medium — No sitemap-only orphan URLs
  • sitemap_broken_urls high — No broken URLs in sitemap
  • sitemap_redirect_urls medium — No redirect URLs in sitemap
  • sitemap_robots_conflict high — No sitemap/robots.txt conflicts
  • sitemap_image low — Sitemap includes image entries
  • sitemap_video info — Sitemap includes video entries
  • redirect_chains medium — No redirect chains detected
  • temporary_redirects low — Temporary redirects are intentional

Links (6 rules)

Internal linking, external links, and broken link detection.

  • internal_links_present medium — Page has internal links
  • internal_links_count low — Internal link count is reasonable
  • internal_links_anchor_text low — Internal links have descriptive anchor text
  • external_links_rel medium — External links have rel attributes
  • broken_internal_links high — No broken internal links
  • broken_external_links medium — No broken external links

Performance (12 rules)

Page size, response time, compression, and resource optimization.

  • page_size medium — Page size is within limits
  • html_size medium — HTML document size is reasonable
  • ttfb high — Time to First Byte is fast
  • compression medium — Response uses compression
  • js_file_count medium — JavaScript file count is reasonable
  • css_file_count medium — CSS file count is reasonable
  • render_blocking high — No render-blocking resources
  • total_requests medium — Total requests are optimized
  • cache_headers low — Cache headers are configured
  • broken_scripts high — All script files are reachable
  • broken_stylesheets high — All CSS files are reachable
  • broken_images medium — All images are reachable

Structured Data (3 rules)

JSON-LD, breadcrumbs, and schema.org validation.

  • jsonld_exists medium — Page has JSON-LD structured data
  • jsonld_valid high — JSON-LD is valid JSON
  • jsonld_breadcrumb low — BreadcrumbList schema is present

Security (9 rules)

HTTPS, security headers, and secure configuration.

  • uses_https critical — Page uses HTTPS
  • mixed_content high — No mixed content detected
  • hsts_header medium — HSTS header is present
  • security_xcto low — X-Content-Type-Options header
  • security_xfo low — X-Frame-Options header
  • security_csp low — Content-Security-Policy header
  • security_referrer low — Referrer-Policy header
  • security_permissions low — Permissions-Policy header
  • security_server_disclosure low — Server version not disclosed

Accessibility (5 rules)

Language, viewport, character encoding, and ARIA landmarks.

  • html_lang high — HTML has lang attribute
  • viewport_meta high — Viewport meta tag is present
  • charset_meta medium — Character encoding declared
  • empty_links medium — Links have accessible text
  • aria_landmarks low — Page uses ARIA landmarks

Mobile (2 rules)

Viewport configuration for mobile devices.

  • mobile_viewport critical — Viewport meta tag exists
  • mobile_viewport_config medium — Viewport uses device-width config

URL Structure (6 rules)

URL format, length, and special characters.

  • url_lowercase low — URL uses lowercase
  • url_hyphens low — URL uses hyphens for separation
  • url_length low — URL length is reasonable
  • url_special_chars low — URL has no special characters
  • url_double_slash medium — URL has no double slashes
  • url_parameters low — URL parameters are minimal

Internationalization (2 rules)

Hreflang validation for multi-language sites.

  • hreflang_valid medium — Hreflang tags are valid
  • hreflang_x_default low — Hreflang x-default is present

E-E-A-T (6 rules)

Experience, Expertise, Authoritativeness, and Trustworthiness signals.

  • eeat_author medium — Content has author attribution
  • eeat_copyright low — Page has copyright notice
  • eeat_about_page medium — Site has an About page
  • eeat_contact_page medium — Site has a Contact page
  • eeat_privacy_policy medium — Site has a Privacy Policy
  • eeat_terms low — Site has Terms of Service

Duplicate Content (3 rules)

Detects duplicate titles, descriptions, and body content across pages.

  • duplicate_content_body high — No duplicate body content
  • duplicate_titles medium — No duplicate page titles
  • duplicate_descriptions medium — No duplicate meta descriptions

  • geo_crawler_perplexitybot high — PerplexityBot is allowed
  • geo_crawler_claude-searchbot high — Claude-SearchBot is allowed
  • geo_crawler_google-extended high — Google-Extended is allowed
  • geo_crawler_applebot high — Applebot is allowed
  • geo_crawler_block_gptbot low — GPTBot (training) is blocked
  • geo_crawler_block_ccbot low — CCBot (training) is blocked
  • geo_llms_txt_exists medium — llms.txt file exists
  • geo_llms_full_txt low — llms-full.txt file exists
  • geo_llms_txt_h1 medium — llms.txt has H1 heading
  • geo_llms_txt_blockquote medium — llms.txt has blockquote summary
  • geo_llms_txt_sections low — llms.txt has H2 sections
  • geo_llms_txt_links low — llms.txt has markdown links
  • geo_entity_org_schema high — Organization schema is present
  • geo_entity_social medium — Social profile links are present
  • geo_citability_statistics medium — Content has citable statistics
  • geo_citability_faq low — FAQ schema is present
  • geo_citability_tables low — Content has data tables
  • geo_citability_lists low — Content has structured lists
  • geo_citability_question_headings medium — Content has question-style headings
  • geo_ai_descriptive_headings medium — Headings are descriptive for AI
  • geo_ai_freshness medium — Content has freshness indicators
  • geo_ai_semantic_html medium — Page uses semantic HTML elements
