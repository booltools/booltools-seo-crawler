Getting Started

Set up and run the Booltools SEO Crawler locally in a few minutes.

Prerequisites

  • Go 1.21 or later
  • Node.js 22 or later (for the frontend)
  • npm (comes with Node.js)

Installation

1. Clone the repository

git clone https://github.com/booltools/booltools-seo-crawler.git
cd booltools-seo-crawler

2. Install Go dependencies

go mod tidy

3. Install frontend dependencies

cd web
npm install
cd ..

Running in Development

Start the backend

make dev
# or: go run ./cmd/server

The API server starts at http://localhost:8080.

Start the frontend

cd web
npm run dev

The web interface is available at http://localhost:4321.

Building for Production

Build the backend

make build
# Binary output: bin/booltools-seo-crawler

Build the SDK CLI

make build-sdk
# Binary output: bin/seo-crawler

Build the frontend

cd web
npm run build

Docker Deployment

docker compose up --build

The app will be available at http://localhost:4321 with the API at http://localhost:8080.

Running Tests

make test
# or: go test ./... -v -count=1

Troubleshooting

Port 8080 already in use
Another process is using port 8080. Stop it or set the PORT environment variable to use a different port.
Node.js version error during frontend build
Astro requires Node.js 22 or later. Run node --version to check and update if needed using nvm install 22.
Go module download fails
Ensure you have internet access and Go 1.21+ installed. Run go version to verify. If behind a proxy, set GOPROXY appropriately.

Next Steps
