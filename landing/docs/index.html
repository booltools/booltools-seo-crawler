Welcome to the Booltools SEO Crawler documentation. This tool analyzes websites for 130+ SEO and GEO checks, providing actionable reports with fix instructions.

What is Booltools SEO Crawler?

A free, open-source website auditor built with Go and Astro. It covers:

  • 15 audit categories — from technical SEO to AI search readiness
  • 130+ individual rules — each with severity levels and fix instructions
  • Real-time analysis — see issues appear live as pages are crawled
  • CI/CD integration — catch regressions before production
