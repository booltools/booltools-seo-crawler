Frequently Asked Questions

What makes this different from other SEO tools?
Booltools SEO Crawler is 100% free, open-source, and includes 24 GEO (Generative Engine Optimization) rules for AI search readiness that most paid tools don't cover. It also runs locally — your data never leaves your machine.
Can I use it in my CI/CD pipeline?
Yes. The SDK is a standalone binary that exits with code 1 when issues are found above your configured threshold. It's available as a GitHub Action on the Marketplace.
Does it support JavaScript-rendered pages?
The crawler fetches server-rendered HTML. For SPAs that rely on client-side rendering, use the static mode with your pre-built output directory, or crawl against a running dev server.
How many pages can it crawl?
By default up to 1,000 pages per domain. You can increase this with the --max-pages flag or the web UI settings. The crawler respects robots.txt and rate-limits itself to avoid overloading target servers.
