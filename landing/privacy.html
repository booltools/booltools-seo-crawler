Privacy Policy

Last updated: June 22, 2026

Overview

Booltools SEO Crawler is a free, open-source tool that runs entirely on your own infrastructure. We are committed to transparency about how data is handled when using our software and this website.

Self-Hosted Software

When you run the Booltools SEO Crawler locally or on your own servers:

  • No data is sent to us. The application runs entirely on your machine.
  • Crawl results are stored in a local SQLite database on your filesystem.
  • No analytics, telemetry, or tracking is included in the software.
  • No accounts or registration are required.

This Website

This documentation website (booltools.github.io) is hosted on GitHub Pages. GitHub may collect basic access logs (IP address, browser type, referring page) as described in the GitHub Privacy Statement.

  • We do not use cookies on this website.
  • We do not use third-party analytics services.
  • We do not collect personal information through this website.
  • Google Fonts are loaded from Google's CDN — see Google's Privacy Policy.

GitHub Action / Marketplace

When using the Booltools SEO Crawler GitHub Action:

  • The action downloads a pre-built binary from GitHub Releases and runs it in your CI environment.
  • No data is transmitted to external servers — analysis happens entirely within your GitHub Actions runner.
  • Results are printed to stdout and optionally written to a local file.

Open Source

The entire source code is publicly available at github.com/booltools/booltools-seo-crawler under the MIT License. You can inspect, audit, and modify the code to verify these privacy claims.

Data Retention

Since the software runs locally on your infrastructure, you have complete control over data retention. Crawl results are stored in a SQLite database file on your local filesystem. You can delete this file at any time to remove all stored audit data. No backups are kept by us, and no data is replicated to any external service.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected by updating the "Last updated" date at the top. Since this is an open-source project, all changes are tracked in the public Git history.

Contact

For questions about this privacy policy, please open an issue on the GitHub repository.
